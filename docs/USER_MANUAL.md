# LiveFake 説明書

実況向け日本語掲示板ブラウザ LiveFake の使い方をまとめた説明書です。  
5ch.io / したらば / JPNKN の3サイトに対応しています。

---

## 1. はじめに

### 1.1 LiveFake とは

- 実況に特化した Windows 専用の掲示板ブラウザ
- **Portable 形式**: ZIP を展開するだけで使える。インストール不要・レジストリ汚染なし
- 設定はすべて EXE 同梱フォルダに保存される

### 1.2 動作環境

- Windows 10 / Windows 11 (64bit)
- WebView2 Runtime (通常 OS にプリインストール済み)

---

## 2. インストールと起動

1. [GitHub Releases](../../releases) から `livefake-win-x64.zip` をダウンロード
2. 任意のフォルダに展開
3. `livefake.exe` をダブルクリックで起動

初回起動時、設定ファイル群が EXE と同じフォルダに自動生成されます。

> **アンインストール**: フォルダごと削除するだけです。

## 3. 画面構成

```
┌──────────────────────────────────────────────┐
│ メニューバー (ファイル/表示/ツール/ヘルプ)    │
├──────────────────────────────────────────────┤
│ ツールバー (URL欄 / 自動更新 / 設定 等)        │
├──────────┬───────────────────────────────────┤
│          │ 板タブバー                          │
│ 板ペイン  ├───────────────────────────────────┤
│ (左)     │ スレタブバー                        │
│          ├───────────────────────────────────┤
│          │ スレッドタイトル + 操作ボタン        │
│ ─────── │ (再読込/書き込み/🔍検索/字幕/TTS等)  │
│ 新着ペイン├───────────────────────────────────┤
│          │ レス表示エリア                       │
│          │                                    │
│          ├───────────────────────────────────┤
│          │ ナビバー (件数 / Top/New/Bottom)     │
├──────────┴───────────────────────────────────┤
│ ステータスバー (URL / 進捗 / 最終更新時刻)     │
└──────────────────────────────────────────────┘
```

各ペインの境界 (薄い縦/横の線) を **ドラッグ** すると幅・高さを調整できます。

## 4. 板を開く

### 4.1 板ツリー (左ペイン)

- カテゴリ名をクリックで展開／折りたたみ
- 板名をクリック → 右側にスレッド一覧を表示
- 上部の検索欄に文字を入れると、板名を絞り込み

### 4.2 お気に入り板

- 板を **右クリック** → 「お気に入りに追加」
- 左ペインの **「Favorite」タブ** に切り替えると一覧表示

### 4.3 外部板の登録 (したらば / JPNKN 等)

- 左ペイン上部の「外部板を追加」ボタン
- URL と板名を入力して登録
- 登録した板は Favorite タブに表示される

### 4.4 板タブ

複数の板を同時に開けます。

- 板を開くたびに **板タブ** が右ペイン上部に追加される
- タブを **ドラッグ** して順序を変更
- タブの **×** で閉じる
- 板タブを **右クリック** で板情報を確認

## 5. スレッド一覧

### 5.1 並び替え

列見出しをクリックでソート。再度クリックで降順/昇順を切替。

| 列 | 内容 |
|----|------|
| 取得 | 取得済みインジケータ |
| ID | スレ ID |
| スレタイ | タイトル |
| レス | レス数 |
| 既読/未読 | 既読・未読件数 |
| 最終取得 | 最後に取得した日時 |
| Since | スレ作成日時 |
| 速度 | 1日あたりレス数の目安 (勢いバー) |

### 5.2 検索

ツールバー上部の検索欄でスレタイを絞り込み (リアルタイム)。  
Enter で履歴に保存。検索履歴ボタン (▽) から呼び出し可。

### 5.3 既読管理

- 一度開いたスレは既読扱い
- 新着があれば未読数列にカウント
- dat 落ちスレはグレー表示 (キャッシュから再表示可能)

### 5.4 右クリックメニュー

- 新タブで開く
- お気に入り登録/解除
- カスタムタイトル設定
- キャッシュから削除

---

## 6. レスを読む

### 6.1 スレタブと表示

- スレッドをダブルクリック → スレタブが追加され右ペインに表示
- スレタブも **ドラッグで並べ替え**、**×で閉じる** が可能
- 同時最大タブ数は設定で変更可 (既定 20)

### 6.2 アンカーポップアップ

- `>>123`、`>>120-130`、`>>1,2,3` 形式の参照をホバー → 該当レスがポップアップ
- ポップアップ内の `>>` も再帰的にホバーでネスト表示

### 6.3 被参照表示

- レス下に `▼3` のように表示 (3 件から参照されている)
- クリックで参照元レスをポップアップ

### 6.4 ID 色分け

- 同一 ID の投稿は同じ色で表示 (15 色から自動割当)
- 書き込み回数: 1 回=デフォルト色、2-4 回=青、5 回以上=赤

### 6.5 画像

- レス内の画像 URL を自動検出 → サムネイル化
- サムネクリック → ライトボックス (別ポップアップ) で拡大表示
- **Ctrl+ホバー** で等倍プレビュー、**Ctrl+ホイール** でズーム

### 6.6 AA (アスキーアート)

- 自動検出して等幅フォントに切替
- 右クリックで AA 表示の手動 ON/OFF も可

### 6.7 自動スクロール

- 自動更新で新着が着いたとき、自動で末尾までスクロール
- ON/OFF はツールバーまたはレス領域の右クリックメニューから切替

## 7. 検索 (レス抽出 / スレ内検索)

スレッドタイトル横の **🔍 (虫眼鏡)** ボタンを押すとプルダウンが開き、2 つのモードから選べます。

### 7.1 レス抽出

- 検索ワードを含むレスだけを表示 (フィルタ)
- 残ったレス内で検索ワードがハイライトされる
- 元に戻すには検索バーを閉じる

### 7.2 スレ内検索

- レスは絞り込まず、検索ワードを含む全レスをハイライト
- バー右側に `現在/総ヒット数` (例 `2/15`) を表示
- **↑ / ↓** ボタンで前/次のヒットレスへジャンプ
- Enter キーでも次のヒットへ移動

両モードとも検索バーを閉じるとハイライトは解除されます (一時ハイライト)。

## 8. 書き込み

### 8.1 通常書き込み

1. レス画面右上の **書き込み (✏️) ボタン**
2. 名前・メール・本文を入力
3. **Ctrl+Enter** または **Shift+Enter** (設定で切替) で送信

### 8.2 引用

- レス番号を **ダブルクリック** → 書き込み欄に `>>N` が自動挿入
- レスを右クリック → 「ここにレス」「名前付き引用」

### 8.3 プレビューと履歴

- 書き込みプレビューで送信前に確認
- 書き込み履歴は最大 50 件 (メニュー → ツール → 書き込み履歴)

### 8.4 新スレ立て

- 板を開いた状態で「スレ立て」メニュー
- タイトルと本文を入力して送信
- ダイアログサイズはドラッグで調整、サイズは自動保存

### 8.5 認証 (BE / UPLIFT / どんぐり)

- ステータスバーから BE ログイン/ログアウト切替
- ログイン状態の Cookie は自動管理 (`cookies.json`)
- Cookie の `Be3M`, `Be3D`, `sid` は安全に永続保存

## 9. NG フィルタ

### 9.1 種類

| 種別 | 対象 |
|------|------|
| ワード NG | レス本文 |
| ID NG | 投稿者 ID |
| 名前 NG | 投稿者名 |
| スレタイ NG | スレッドタイトル |

### 9.2 NG モード

- **非表示**: レス全体を非表示
- **画像のみ非表示**: テキストは表示し画像だけブロック

### 9.3 登録方法

- 設定 → NG タブで一覧管理 (追加/削除)
- レス上で右クリック → 「○○を NG ワードに追加」「NG ID に追加」等
- 正規表現は `/パターン/` 形式で指定

## 10. ハイライト

15 色パレットから色を選んで強調表示できます。

| 種別 | 効果 |
|------|------|
| テキストハイライト | 本文中のキーワードを背景色で強調 |
| 名前ハイライト | 投稿者名を背景色で強調 |
| ID ハイライト | ID の文字色を変更 (当日分のみ自動リセット) |

- 右クリックメニューから即時追加/解除
- 設定 → ハイライト で全体管理
- 字幕ウィンドウにも同じ色が同期される

## 11. 実況機能

### 11.1 字幕ウィンドウ

レス画面右上の **字幕ボタン** で別ウィンドウとして開きます。

- ドラッグで自由移動
- 「中央に戻す」で初期位置にリセット
- 透明度 (0.1 〜 1.0)、本文/メタ情報のフォントサイズを個別設定
- Always on Top のトグル
- メインウィンドウを閉じると自動クローズ

### 11.2 新着ペイン

スレ表示の上にサブパネルとして表示。

- 右上の ▽ で非表示、レス画面右上のボタンで再表示
- 高さは下端のリサイザでドラッグ調整
- 自動更新で取得した新着レスを順次流す

### 11.3 音声読み上げ (TTS)

3 つのエンジンに対応。設定 → 読み上げ で選択。

| エンジン | 必要なもの |
|---------|-----------|
| **SAPI** | 不要 (Windows 標準) |
| **棒読みちゃん** | 棒読みちゃん本体を起動 |
| **VOICEVOX** | VOICEVOX エンジンを `http://127.0.0.1:50021` で起動 |

- 自動更新が有効なときの新着レスを自動再生
- レス番号を右クリック → 「このレスから読み上げ」で手動再生
- 文字数制限を超えるレスは省略 (10〜300 文字、無制限可)

## 12. 画像と画像 URL 置換

### 12.1 画像保存

- ライトボックス → 「保存」 (デフォルトのダウンロードフォルダに保存)
- 保存先フォルダは設定で変更可

### 12.2 画像アップロード

- メニュー → ツール → 画像アップロード (`tadaup.jp`)
- アップロード後の URL を書き込み欄に貼付

### 12.3 ImageViewURLReplace.txt

- TSV 形式: `検索正規表現<TAB>置換文字列<TAB>リファラ`
- ニコニコ動画のサムネ表示や、Twitter/X の画像CDN置換に活用
- ファイルは EXE と同じフォルダに自動生成

## 13. 設定パネル

ツールバーの **歯車アイコン** から開きます。タブで切替。

| タブ | 主な項目 |
|------|--------|
| 表示 | テーマ (ライト/ダーク)、フォント、サイズ、自動更新間隔、自動スクロール、タブ上限、画像サイズ制限 |
| 書き込み | 送信ショートカット (Ctrl+Enter / Shift+Enter)、書き込み文字サイズ |
| 読み上げ | エンジン選択、SAPI/棒読みちゃん/VOICEVOX 詳細設定、文字数制限 |
| 字幕 | 透明度、本文/メタフォントサイズ、Always on Top、位置リセット |
| プロキシ | タイプ (HTTP/SOCKS5)、ホスト・ポート、認証 (パスワードは DPAPI 暗号化) |
| NG | NG フィルタの追加・削除 |
| ハイライト | テキスト/名前/ID ハイライトの追加・削除 |
| 情報 | バージョン、更新チェック |

## 14. キーボード操作

### 14.1 レスエリア (フォーカス時)

| キー | 動作 |
|------|------|
| ↑ / ↓ | 60px ずつスクロール |
| PageUp / PageDown | 1 ページ分スクロール |
| Home / End | 先頭 / 末尾へ |

### 14.2 全体

| キー | 動作 |
|------|------|
| Esc | メニュー / ポップアップ / ホバープレビューを閉じる |
| Ctrl+Enter / Shift+Enter | 書き込み送信 (設定で切替) |

> 個別の入力欄では入力欄独自の挙動が優先されます (検索欄の Enter で履歴保存など)。

## 15. 設定ファイル (Portable)

EXE と同じフォルダに自動生成されます。バックアップや別マシンへの移行はフォルダごとコピーで完結。

| ファイル | 内容 |
|---------|------|
| `settings.ini` | アプリ全般設定 (フォント・自動更新・TTS 等) |
| `favorites.json` | お気に入り (板/スレ) |
| `ng-settings.json` | NG フィルタ |
| `text-highlights.json` | テキスト/名前ハイライト |
| `id-highlights.json` | ID ハイライト (当日分) |
| `cookies.json` | 認証 Cookie |
| `bbs-menu.json` | BBS Menu キャッシュ |
| `bookmarks.json` | レスのしおり位置 |
| `scroll-positions.json` | スレごとのスクロール位置 |
| `read-status.json` | 既読状態 |
| `thread-history.json` | スレ訪問履歴 (カスタムタイトル含む) |
| `name-history.json` | 投稿者名の履歴 |
| `search-history.json` | 検索履歴 |
| `my-posts.json` | 自分の投稿レス番号 |
| `upload-history.json` | 画像アップロード履歴 |
| `proxy-settings.json` | プロキシ設定 |
| `layout-prefs.json` | ペイン幅などレイアウト |
| `session-tabs.json` / `session-board-tabs.json` | 開いていたタブ |
| `board-categories.json` | 板カテゴリのキャッシュ |
| `expanded-categories.json` | 板ツリーの展開状態 |
| `thread-fetch-times.json` | スレごとの最終取得時刻 |
| `new-thread-dialog-size.json` | スレ立てダイアログサイズ |
| `external-boards.json` | 外部板登録 |
| `ImageViewURLReplace.txt` | 画像 URL 置換ルール |
| `thread-cache.db` | スレッドレスキャッシュ (SQLite) |
| `eventlog/` | アプリログ (既定 7 日保持) |

## 16. よくある質問

### Q. 起動するとすぐ閉じてしまう

WebView2 Runtime が入っていない可能性があります。Windows Update を最新にするか、Microsoft の WebView2 ランタイムを手動でインストールしてください。

### Q. 板タブやスレタブのドラッグで並べ替えができない

タブの **タイトル部分** をドラッグしてください。閉じるボタン (×) の上から始めるとドラッグになりません。6px 以上動かすとドラッグ判定になります。

### Q. 自動更新が効かない

ツールバーの **自動更新トグル** が ON になっているか確認。OFF だと定期取得しません。間隔は 15〜300 秒で設定可。

### Q. 字幕ウィンドウが表示されない

メインウィンドウのタイトルバー右側 / レス画面右上の **字幕アイコン** で開閉します。位置がモニタ外にずれた場合は設定 → 字幕 → 「中央に戻す」を実行。

### Q. 設定や履歴を別の PC に移したい

EXE と同じフォルダ内のファイル一式 (上記 15 章) をそのままコピーすれば移行完了です。`thread-cache.db` も含めれば過去ログ含めて完全移行できます。

### Q. パスワードはどう保護されている?

プロキシのパスワードは Windows DPAPI で暗号化されてから保存されます (同一 Windows ユーザーアカウントでのみ復号可能)。BE/UPLIFT/どんぐりの認証情報は Cookie として保存されます。

### Q. 過去のバージョンに戻したい

ZIP ファイルを古いバージョンと差し替えるだけで OK。設定ファイルは互換性が保たれます。
